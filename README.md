# Поисковая система на Spring Boot

Это учебный проект по созданию **поисковой системы**, способной индексировать сайты, извлекать леммы (нормальные формы слов) и выполнять полнотекстовый поиск с ранжированием результатов по релевантности.

Проект реализует:
- Парсинг HTML-страниц через Jsoup.
- Индексацию сайтов с обходом ссылок.
- Морфологический анализ текста (лемматизацию) для русского и английского языков.
- Поиск по запросу с учётом всех слов в запросе.
- Ранжирование результатов по абсолютной и относительной релевантности.
- REST API для управления индексацией и выполнения поиска.

---

##  Стек технологий

- **Язык**: Java 17
- **Фреймворк**: Spring Boot 2.7.1
- **СУБД**: PostgreSQL
- **Морфология**: Apache Lucene Morphology (`RussianLuceneMorphology`)
- **Парсинг HTML**: Jsoup
- **Управление зависимостями**: Maven
- **Миграции БД**: Liquibase
- **Веб-интерфейс**: Thymeleaf (для главной страницы)
- **API**: REST, JSON
- **Сборка и запуск**: IntelliJ IDEA / Maven

##  Инструкция по локальному запуску

### 1. Требования

Перед началом убедитесь, что у вас установлено:

- ✅ **JDK 17** — [скачать](https://adoptium.net/temurin/releases/?version=17)
- ✅ **PostgreSQL** — [скачать](https://www.postgresql.org/download/)
- ✅ **Maven 3.6+** — (или используйте встроенный в IntelliJ IDEA / VS Code)
- ✅ Любой текстовый редактор или IDE (рекомендуется **IntelliJ IDEA**)
- ✅ Доступ в интернет (для загрузки зависимостей)

---

### 2. Настройка базы данных PostgreSQL

#### Шаг 1: Запустите PostgreSQL сервер

Убедитесь, что сервер PostgreSQL запущен на вашем компьютере.

#### Шаг 2: Создайте базу данных и пользователя

Подключитесь к PostgreSQL через командную строку, DBeaver или другую утилиту и выполните:

```sql
CREATE DATABASE search_engine;
```

#### 3. Настройка конфигурации приложения
"Используйте postgres данные и ваш пароль в application.yml."
```yaml
  datasource:
    username: postgres
    password: password
    url: jdbc:postgresql://localhost:5432/search_engine
    driver-class-name: org.postgresql.Driver
```
#### 4. Сборка и запуск проекта
Через терминал:
```bash
cd /path/to/project
mvn clean compile
mvn spring-boot:run
```
Через IntelliJ IDEA:

```
-Откройте проект как Maven-проект.
-Дождитесь загрузки зависимостей.
-Запустите Application.java.

→ Приложение доступно по адресу: http://localhost:8080
```
#### 5. Первый запуск

При старте Liquibase создаст таблицы.
```
Откройте http://localhost:8080 — должна открыться главная страница.
```
